# StarTech Selenium Automation Practice Project

A web automation project for practice built with **Selenium WebDriver** and **JavaScript (Node.js)** for automating user interactions on the Star Tech website.

## Features

- Product search automation
- Login automation
- Product navigation
- Search product verify
- Add to Cart automation
- Checkout 

## Tech Stack

- JavaScript (Node.js)
- Selenium WebDriver
- ChromeDriver
- npm

## Project Structure

```
startech-selenium-automation/
│── pages/

```

## Installation

Clone the repository:

```bash
git clone https://github.com/ankandas101/startech-selenium-automation.git
```

Move into the project directory:

```bash
cd startech-selenium-automation
```

Install dependencies:

```bash
npm install
```

## Run Tests

Run a specific test:

```bash
node test.js
```

Or run another test:

```bash
node tests/searchProduct.test.js
```

## Dependencies

```bash
npm install selenium-webdriver
```
